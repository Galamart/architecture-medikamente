## Оценка узких мест при миграции

### 1. Ключевые процессы и компоненты, затрагиваемые при миграции:
- Базы данных - Перенос из Excel/1C в PostgreSQL
- Интеграции - API-интеграции между сервисами, с 1С, с платёжным шлюзом, c лабораторией
- Инфраструктура - Переход с локального сервера на Kubernetes + облако
- Бизнес-логика - Реализация правил записи, доступа, оплаты в новых сервисах
- Каналы связи - Защита транспорта данных между клиентом, бэкендом, партнёрами
- Пользователи - Обучение персонала работе с новыми инструментами
